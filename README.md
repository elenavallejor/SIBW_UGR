# ALTURAS – Portal de Actividades y Excursiones

Este proyecto forma parte de la asignatura Sistemas de Información Basados en Web (SIBW) de la Universidad de Granada.  
Consiste en una aplicación web que permite la gestión y visualización de actividades, permitiendo a los usuarios consultar eventos, registrarse, comentar y, en el caso de administradores, administrarlos y moderar su contenido.

---

## Descripción General

La web ofrece un listado de actividades y permite acceder a su información completa.  
El sistema dispone de funciones de autenticación de usuarios y de un panel de control para la gestión interna.

El propósito es simular el funcionamiento de un portal real de excursiones o actividades sociales.

---

## Funcionalidades Principales

### Usuarios Registrados
- Registro e inicio de sesión.
- Consulta detallada de actividades.
- Búsqueda de actividades por nombre o por hashtags.
- Publicación de comentarios en las actividades.

### Administradores
- Añadir nuevas actividades.
- Editar o archivar actividades existentes.
- Eliminar actividades.
- Moderar y eliminar comentarios.
- Gestión de usuarios y permisos.
- Censura automatica de palabras prohibidas
